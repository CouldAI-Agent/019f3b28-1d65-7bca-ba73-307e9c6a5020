# पावर बीआई (Power BI) डैशबोर्ड क्लोन

यह एक फ़्लटर (Flutter) आधारित एनालिटिक्स और डेटा विज़ुअलाइज़ेशन डैशबोर्ड है, जिसे Power BI के अनुभव को ध्यान में रखकर बनाया गया है। इसमें इंटरेक्टिव चार्ट्स और KPI (Key Performance Indicator) कार्ड्स का उपयोग किया गया है।

## मुख्य विशेषताएँ (Features)
- **रेस्पॉन्सिव डिज़ाइन (Responsive Design):** मोबाइल, टैबलेट और डेस्कटॉप तीनों प्लेटफॉर्म पर बेहतरीन लेआउट।
- **KPI मेट्रिक्स:** कुल रेवेन्यू, एक्टिव यूज़र्स और बाउंस रेट जैसे ज़रूरी आँकड़े देखने के लिए कार्ड्स।
- **चार्ट्स:** `fl_chart` पैकेज का उपयोग करके लाइन चार्ट (Revenue Trend) और बार चार्ट (Sales by Region) को प्रदर्शित किया गया है।
- **कस्टम नेविगेशन:** डेस्कटॉप पर साइडबार नेविगेशन और मोबाइल पर ड्रॉअर (Drawer) मेन्यू।

## तकनीक (Tech Stack)
- Flutter (क्रॉस-प्लेटफॉर्म UI के लिए)
- Dart
- `fl_chart` (डेटा विज़ुअलाइज़ेशन और चार्ट्स के लिए)

## कैसे चलाएं (How to Run)
इस प्रोजेक्ट को रन करने के लिए नीचे दिए गए कमांड का उपयोग करें:
```bash
flutter pub get
flutter run
```
